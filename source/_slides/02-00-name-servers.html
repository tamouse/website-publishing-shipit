What is a Name Server?

Name servers provide the glue between your domain name and your IP address.

IP addresses can change over time, while it's rare your domain name will.

Domain: tamouse.org IP Address: 45.55.188.49
Domain: google.com IP Address: 172.217.4.110
Domain: gdiminneapolis.com IP Address: 192.185.79.136

{{page.title}}

Linking Domain to Host

Your hosting and domain name may not come from the same company. The nameservers link your domain to your hosting.

  • Domain Name: MyCoolWebsite.com
  • IP Address: 199.167.200.163
  • Name Server: ns1.gdiminneapolis.com
